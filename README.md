# Projeto de jogo info
Esse projeto é um jogo que foi desenvolvido ao decorrer dos anos de 2025/2026 no Colégio Tecnico Industrial de Santa Maria para o projeto "DESENVOLVIMENTO DE JOGOS ELETRÔNICOS COMO FORMA DE APRENDIZAGEM PARA ALUNOS DO CURSO TÉCNICO EM INFORMÁTICA PARA INTERNET"

O objetivo foi estudar e encontrar maneiras de criar jogos e consequentemente como eles podem influênciar no aprendizado de programação.

Desenvolvido usando Godot 4.5
